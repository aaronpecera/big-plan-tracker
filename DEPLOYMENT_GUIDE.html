3. ğŸƒ MongoDB Atlas Setup

Step 3.1: Create MongoDB Cluster

  1. Go to MongoDB Atlas
  2. Sign up for a free account
  3. Create a new project called "BigPlanTracker"
  4. Click "Build a Database"
  5. Choose "M0 Sandbox" (FREE tier)
  6. Select your preferred region
  7. Name your cluster "big-plan-tracker"

Step 3.2: Configure Database Access

  1. Go to "Database Access" in the left menu
  2. Click "Add New Database User"
  3. Choose "Password" authentication
  4. Username: bigplanuser
  5. Generate a secure password (save it!)
  6. Database User Privileges: "Read and write to any database"

Step 3.3: Configure Network Access

  1. Go to "Network Access" in the left menu
  2. Click "Add IP Address"
  3. Click "Allow Access from Anywhere" (0.0.0.0/0)
  4. Click "Confirm"

Step 3.4: Get Connection String

  1. Go to "Databases" in the left menu
  2. Click "Connect" on your cluster
  3. Choose "Connect your application"
  4. Select "PHP" and version "1.15.0 or later"
  5. Copy the connection string (looks like this):
mongodb+srv://bigplanuser:<password>@big-plan-tracker.xxxxx.mongodb.net/?retryWrites=true&w=majority
Important: Replace <password> with your actual database password!
